	DEVOLUÇÕES

	ENDEREÇO
-	(1) ESTOQUE PARA 1 ENDEREÇO, CADA CADASTRO REGISTRARÁ APENAS UM ENDEREÇO, SENDO POSSÍVEL CRIAR VARIOS ESTOQUES, PORÉM UM ENDEREÇO PARA CADA ESTOQUE.

	TELEFONE
-	(N) ESTOQUES PARA N DIVERSOS TELEFONES, DIVERSOS ESTOQUES EM CADA CADASTRO DE CONTROLADOR DO ESTOQUE PODE TER DIVERSOS TELEFONES. 


	PRODUTOS
-	(N) ESTOQUES PARA N DIVERSOS PRODUTOS, DIVERSOS ESTOQUES POSSUI DIVERSOS PRODUTOS QUE SERÁ SEPARADOS POR CATEGORIA. 


	PEDIDOS
-	(N) ESTOQUES PARA N DIVERSOS PEDIDOS, DIVERSOS ESTOQUES PODE TER DIVERSOS PEDIDOS NÃO É OBRIGATÓRIO.
	


	CONTRATO
